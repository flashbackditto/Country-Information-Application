Quick Country Information

This application is for anyone curious about the capital, population, currency, size or weather of any country. All UN recognized countries - as well as a few that aren't - are included. Search for specific countries in the search bar or display all of the countries by selecting one of the sorting options. If you're interested in more information about a particular country, clicking on any individual country's card will open the relevant wiki page in a new tab.

General capital, population, currency, and size information in this application is provided by restcountries.eu API and each country's capital weather information is accessed through the api.openweathermap.org API. These two API's being free is quite generous. However, in life you often get what you pay for, in this case, that statement rears it's head as a limit on API requests per minute. If weather information is not currently being displayed it is likely other users have temporarily locked API access by making more requests than the free access allows per minute, this should only last a few minutes before normal service continues.
