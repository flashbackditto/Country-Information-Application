APIs

General capital, population, currency, and size information in this application is provided by restcountries.eu API and each country's capital weather information is accessed through the openweathermap.org API. These two API's being free is quite generous. However, in life you often get what you pay for, in this case, that statement rears it's head as a limit on API requests per minute. If weather information is not currently being displayed it is likely other users have temporarily locked API access by making more requests than the free access allows per minute, this should only last a few minutes before normal service continues.
